About This Project

The A/L Physics Learning Assistant is an AI-powered educational tool designed specifically for Sri Lankan Advanced Level students studying Physics. This application leverages artificial intelligence to provide personalized learning experiences, practice questions, exam tips, study plans, and concise notes.

Features

  • Interactive learning with step-by-step explanations
  • Practice questions with detailed solutions
  • Strategic exam tips and techniques
  • Customized study plans
  • Concise revision notes
  • Support for both English and Sinhala languages

Technologies Used

  • Python with Flask framework
  • HTML, CSS, and JavaScript
  • Bootstrap for responsive design
  • Google Gemini AI for content generation
  • Deployed on cloud infrastructure

Developer Information

This application was developed by Chamika, a passionate developer focused on creating educational tools to help students succeed in their academic journey.
